# JSoft

### Requirements
* [Java](https://openjdk.org)  
* [Maven](https://maven.apache.org)  
* [JUnit](https://junit.org)

#### Unit tests
* Windows, MacOS/Linux
```
mvn test
```
#### Build application
* Windows, MacOS/Linux
```
mvn clean package
```
#### Run application
* Windows, MacOS/Linux
```
java -jar jsoft-x.x.x.jar
```
